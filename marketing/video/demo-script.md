# Hivemind Demo Video Script (60-90 seconds)

## Target: X/Twitter, Reddit, YouTube Shorts, LinkedIn

---

## Shot 1: The Problem (0-12s)
**Visual:** Split screen — left side shows a developer typing "continue" repeatedly in Claude Code, right side shows a clock spinning fast.

**Narration/Text overlay:**
> "You asked Claude Code to build a feature. 45 minutes later, you're still babysitting."

**Text on screen:** "Sound familiar?"

---

## Shot 2: The Solution (12-20s)
**Visual:** Terminal opens. User types one line:

```
"Add JWT authentication with a login page and protected routes"
```

**Text overlay:** "One prompt. That's it."

---

## Shot 3: The Team Assembles (20-35s)
**Visual:** Dashboard view — DAG appears with nodes lighting up one by one:
- PM Agent → creates the plan
- Frontend Dev, Backend Dev, Database Expert → start working IN PARALLEL (highlight this)
- Arrows show artifact flow between agents

**Text overlay:** "A full engineering team. Working in parallel."

---

## Shot 4: Real-Time Dashboard (35-50s)
**Visual:** Quick cuts of the dashboard:
1. Live streaming of agent output
2. DAG progress visualization
3. Agent cards showing status (Working → Done)
4. File diff view
5. Mobile view (phone mockup)

**Text overlay:** "Monitor everything. Even from your phone."

---

## Shot 5: Self-Healing (50-58s)
**Visual:** An agent card turns red (failed). Then automatically: retry icon → agent restarts → turns green.

**Text overlay:** "Self-healing. When an agent fails, the system fixes it."

---

## Shot 6: The Result (58-70s)
**Visual:** IDE opens. New files appear. Tests pass (green checkmarks). Git log shows clean commit.

**Text overlay:** "Tested. Reviewed. Committed. Open your IDE — it's already there."

---

## Shot 7: CTA (70-80s)
**Visual:** Dark background. Hivemind logo. Terminal with the install command.

**Text on screen:**
```
npx create-hivemind@latest
```

**Tagline:** "One prompt. A full AI engineering team. Go lie on the couch."

**GitHub stars counter animating up.**

---

## Production Notes

### Style
- Dark mode throughout (matches landing page aesthetic)
- Purple/blue gradient accents
- Clean, minimal — no flashy transitions
- JetBrains Mono for code, Inter for UI text
- Smooth zoom-in/zoom-out transitions between shots

### Audio
- Lo-fi electronic background track (subtle, not distracting)
- Keyboard typing sounds for terminal shots
- Subtle "whoosh" for transitions
- Satisfying "ding" when tests pass

### Format
- 16:9 for YouTube/LinkedIn
- 9:16 crop for X/Instagram Reels
- Both versions should be produced

### Tools for Production
- Screen recording of actual Hivemind dashboard
- After Effects or Motion for text overlays
- Alternatively: AI video generation for concept shots + real dashboard recording
