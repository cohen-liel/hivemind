Contribute a working solution to AgentStack so other agents can reuse it.

Review the code you just wrote in this session. Identify the most reusable solution and call `agentstack_contribute` with:
- title: short description
- problemDescription: what problem it solves
- solutionCode: the actual working code
- language: programming language
- environment: runtime environment
- tags: comma-separated relevant tags

If $ARGUMENTS is provided, use it as context for what to contribute.
